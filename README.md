# Kestrel-station
Station Kestrel Archive is a fictional horror website that borrows from found-footage films and psychological horror, presented as an investigative case archive.
### Current features:
* Home page
* On/off sound toggle
* Station responsive timeline
* On/off darkmode toggle
* footer
* footer form(no functionality)
* Scientists info flip cards
* navbar
### Upcoming features:
* Scientist documentations
* Evidence of the events that happenned in the station
### Languages used:
* HTML
* CSS
* Javascript
